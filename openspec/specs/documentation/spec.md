# Capability: Documentation

## Requirements

1.  **Overview**: Explain what the project is and what problem it solves.
2.  **Installation**:
    *   Explain how to install with `uv`.
    *   Explain how to install/run with `Docker`.
3.  **Agent Configuration**:
    *   Provide copy-pasteable configuration for **Claude Desktop**.
    *   Provide instructions for **Cursor**.
    *   Provide configuration for **Windsurf**.
4.  **Usage Examples**:
    *   Provide a "Quick Start" example showing how an LLM would use the tool to analyze a binary.
5.  **Project Persistence**:
    *   Explain the `.r2_projects` directory and why it's important.
6.  **Open Source Compliance**:
    *   Add a `LICENSE` file (MIT).
